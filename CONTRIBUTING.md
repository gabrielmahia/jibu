# Contributing to m-pesa-rails

## Bug reports and feature requests

Open an issue. Include:
- Python version
- SDK version (`pip show m-pesa-rails`)
- Sandbox or production
- The error message or unexpected behaviour

Do not include credentials, API keys, or customer phone numbers in issues.

## Security vulnerabilities

See [SECURITY.md](SECURITY.md). Email directly — do not open a public issue.

## Code contributions

This repository does not accept pull requests at this time.
Open an issue to discuss proposed changes first.

Contact: contact@aikungfu.dev
